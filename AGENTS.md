# Agent Instructions

This repository is a Docusaurus documentation site for internal Pemrograman Open Source assistant modules covering R, Python, Git/GitHub workflows, and mini projects.

- Package manager: Bun.
- Use Node.js 20 or newer.
- Common commands: `bun install`, `bun run start`, `bun run build`, and `bun run typecheck`.
- Follow [shell workflow](.codex/agents/shell-workflow.md) for local command execution.
- Follow [Docusaurus workflow](.codex/agents/docusaurus-workflow.md) when editing site content, navigation, or configuration.
- See [instruction audit](.codex/agents/instruction-audit.md) for contradictions checked and instructions flagged for deletion.
- See [suggested docs structure](.codex/agents/docs-structure.md) for organizing course content.
