# Modul Asdos Pemrograman Open Source

Selamat datang di modul pendamping untuk praktikum Pemrograman Open Source.
Materi ini disiapkan untuk membantu mahasiswa memahami alur kerja open source
sambil mempraktikkan analisis data dengan **R** dan **Python**.

## Tujuan Pembelajaran

Setelah mengikuti modul ini, mahasiswa diharapkan mampu:

- menyiapkan lingkungan kerja reproducible untuk R dan Python;
- menulis kode yang rapi, bisa dibaca, dan mudah ditinjau;
- mengolah data tabular dasar dengan R dan Python;
- memakai Git dan GitHub untuk kolaborasi praktikum;
- menyelesaikan proyek mini berbasis dokumentasi terbuka.

## Alur Modul

Materi disusun agar bisa dipakai bertahap dalam sesi asistensi:

1. **Persiapan**: instalasi tools, struktur folder, Git, dan kebiasaan kerja.
2. **R**: dasar bahasa, data frame, eksplorasi data, dan visualisasi.
3. **Python**: dasar bahasa, notebook/script, data analysis, dan visualisasi.
4. **Proyek Mini**: latihan akhir dengan rubrik penilaian dan checklist review.

## Cara Memakai Modul

Gunakan sidebar di kiri untuk mengikuti urutan materi. Setiap bagian berisi
konsep, contoh kode, latihan singkat, dan catatan asdos untuk memandu diskusi
di kelas atau respons praktikum.

```bash
# Jalankan dokumentasi secara lokal
bun install
bun run start
```

Untuk penggunaan internal, jalankan situs secara lokal atau bagikan hasil build
ke tim asdos melalui kanal yang sudah disepakati.
