# Pengantar
Selamat datang di modul pendamping untuk praktikum Pemrograman Open Source.
Materi ini disiapkan untuk membantu mahasiswa memahami alur kerja open source
sambil mempraktikkan analisis data dengan **R** dan **Python**.

## Tujuan Pembelajaran
Setelah mengikuti modul ini, mahasiswa diharapkan mampu:
- menyiapkan lingkungan kerja reproducible untuk R dan Python;
- menulis kode yang rapi, bisa dibaca, dan mudah ditinjau;
- mengolah data tabular dasar dengan R dan Python;
- memakai Git dan GitHub untuk kolaborasi praktikum;
