# Proyek Mini

Proyek mini membantu mahasiswa menerapkan alur open source dalam skala kecil:
membuat repository, menulis dokumentasi, melakukan analisis, dan membuka pull
request.

## Pilihan Tema

- Analisis data nilai praktikum anonim.
- Ringkasan dataset publik sederhana.
- Visualisasi tren dari data CSV.
- Perbandingan hasil analisis R dan Python.

## Deliverables

- Repository GitHub berisi `README.md`.
- Folder `data/`, `src/`, dan `reports/`.
- Minimal satu script R atau Python.
- Ringkasan hasil analisis dalam Markdown.
- Pull request dengan deskripsi perubahan.

## Rubrik Ringkas

| Aspek | Bobot |
| --- | ---: |
| Struktur repository | 20% |
| Kualitas kode | 30% |
| Analisis dan visualisasi | 30% |
| Dokumentasi dan kolaborasi | 20% |

## Checklist Review

- Apakah instruksi menjalankan proyek jelas?
- Apakah file data sensitif tidak ikut di-commit?
- Apakah nama variabel mudah dipahami?
- Apakah grafik menjawab pertanyaan analisis?
