# Setup Lingkungan

Bagian ini memastikan mahasiswa punya lingkungan kerja yang konsisten sebelum
masuk ke materi R dan Python.

## Tools Wajib

- **Git** untuk version control.
- **GitHub** untuk menyimpan repository dan membuka pull request.
- **Visual Studio Code** atau editor lain yang nyaman.
- **R** dan **RStudio** untuk materi R.
- **Python 3.11+** untuk materi Python.
- **Quarto** atau **Jupyter Notebook**.

## Struktur Folder Praktikum

Gunakan struktur sederhana berikut agar proyek mudah direview:

```text
nama-proyek/
  data/
    raw/
    processed/
  notebooks/
  src/
  reports/
  README.md
```

## Checklist Awal

- Bisa menjalankan `git --version`.
- Bisa membuat repository GitHub.
- Bisa menjalankan `R --version` atau membuka RStudio.
- Bisa menjalankan `python --version`.
- Bisa membuat file `README.md` dengan deskripsi singkat proyek.

## Catatan Asdos

Pastikan mahasiswa memahami perbedaan file data mentah, data hasil olahan, kode,
dan laporan. Kebiasaan ini membuat proyek praktikum lebih rapi sejak awal.
