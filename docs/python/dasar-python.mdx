# Dasar Python

Python digunakan untuk scripting, otomasi, data analysis, web, dan machine
learning. Pada praktikum ini, fokus awalnya adalah sintaks dasar dan cara
menulis kode yang mudah dibaca.

## Variabel dan List

```python
mata_kuliah = "Pemrograman Open Source"
nilai = [80, 88, 92, 75]

rata_rata = sum(nilai) / len(nilai)
print(rata_rata)
```

## Percabangan

```python
skor = 88

if skor >= 80:
    print("Lulus sangat baik")
elif skor >= 60:
    print("Lulus")
else:
    print("Perlu remedial")
```

## Latihan

1. Buat list nilai praktikum.
2. Hitung rata-rata nilai.
3. Tampilkan status berdasarkan rata-rata tersebut.

## Catatan Asdos

Python sensitif terhadap indentasi. Saat mendampingi mahasiswa, cek indentasi
lebih dulu sebelum membahas error yang lebih kompleks.
