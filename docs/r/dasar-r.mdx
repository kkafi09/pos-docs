# Dasar R

R cocok digunakan untuk eksplorasi data, statistik, visualisasi, dan laporan
reproducible. Pada bagian awal, fokuskan mahasiswa pada objek dasar dan cara
membaca pesan error.

## Variabel dan Vektor

```r
nama <- "Pemrograman Open Source"
nilai <- c(80, 88, 92, 75)

mean(nilai)
max(nilai)
```

## Tipe Data Dasar

```r
angka <- 10
teks <- "data"
logika <- TRUE

class(angka)
class(teks)
class(logika)
```

## Latihan

1. Buat vektor berisi lima nilai praktikum.
2. Hitung rata-rata, nilai minimum, dan nilai maksimum.
3. Tulis kesimpulan singkat dalam komentar kode.

## Catatan Asdos

Tekankan bahwa R memakai assignment `<-`, tetapi `=` juga sering ditemui.
Untuk praktikum, pilih satu gaya dan gunakan secara konsisten.
